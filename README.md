Bitácora del Bosque Esclerófilo
Este proyecto nace del amor por la naturaleza y la necesidad de proteger nuestro ecosistema. Es una herramienta sencilla diseñada para que cualquier observador pueda registrar la salud de árboles y plantas nativas (como el Quillay), evaluando aspectos clave como la calidad del suelo y la aparición de nuevos brotes. El objetivo es transformar nuestra convivencia con el entorno en un acto de cuidado activo, usando la tecnología para entender mejor la red de vida que nos sostiene.

En cuanto a su funcionamiento, la página permite ingresar datos de observación que se procesan automáticamente para calcular un índice de vitalidad. A través de este ejercicio, puse en práctica los pilares fundamentales de programación, desde cómo recibir información del usuario y realizar cálculos matemáticos, hasta cómo organizar grandes listas de registros en un historial de seguimiento. Es un sistema dinámico que ayuda a visualizar si una especie está en óptimas condiciones o si necesita nuestra ayuda para sobrevivir.
